# Cyber-Security-and-Forensics

This repository contains various scripts, tools, and a mini project related to Cyber Security and Forensics. It includes code for network security, steganography, proxy detection, keylogging, and more.

## Structure

- **attacker.py, receiverotp.py, senderotp.py, userA.py, userB.py, etc.**: Standalone scripts for different security and forensics tasks.

## Notable Scripts
- **honeypot.py**: Simple honeypot implementation
- **keyboardlogging.py**: Keylogger script
- **proxydetection.py**: Detects proxy usage
- **steganosender.py / steganoreceiver.py**: Steganography tools
- **stego_detector.py**: Detects steganography in files
- **dnsresolve.py**: DNS resolution utility
- **echoserver.py**: Echo server for network testing

## How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/SohamMisra/Cyber-Security-and-Forensics.git
   ```
2. Navigate to the project directory and run the desired script:
   ```sh
   python attacker.py
   ```
   (Replace with the script you want to run.)

3. For the mini project, see `CSFMiniProject/README.md` for details.

## Requirements
- Python 3.x
- Additional dependencies may be required for some scripts (see comments in each script or project README).
